## Syscalls

`open(2)`  - opens an fd  
`read(2)`  - reads from an fd  
`write(2)` - writes to a fd  
`close(2)` - removes fd from open-fd table  
`lseek(2)` - adjusts file offset of an fd  
`ioctl(2)` - Perform extra file and device operations

## More Info

See `creat(2)` for an obsolete system call to open and create a
file.
